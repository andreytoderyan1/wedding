# Logo Prompt for HomeQuote

## ChatGPT/DALL-E Prompt

```
Create a professional, modern logo for "HomeQuote" - an HVAC services company. The logo should feature:

- Clean, minimalist design
- Professional and trustworthy appearance
- Incorporate subtle HVAC/home elements (like a house icon, HVAC symbol, or home comfort imagery)
- Modern typography for "HomeQuote" text
- Color scheme: Blue (#3B82F6 or similar) and white, or dark gray/black with blue accents
- Style: Professional business logo, suitable for a home services company
- Format: Horizontal layout preferred
- Background: Transparent or white
- Mood: Trustworthy, professional, modern, approachable

The logo should work well on both light and dark backgrounds. Include the text "HomeQuote" as part of the logo design. Make it suitable for use as a website header logo.
```

## Image Specifications

- **Format**: PNG with transparent background (preferred) or SVG
- **Size**: At least 200x60px (width x height) for web use
- **Aspect Ratio**: Horizontal, approximately 3:1 or 4:1
- **File Name**: `logo.png` or `logo.svg`
- **Location**: `/static/logo.png` or `/static/logo.svg`

## Current Code Location

The logo is referenced in: `src/lib/components/Navigaton.svelte`
- Current path: `/logo.png`
- Update the `src` attribute if you use a different filename

## Quick Steps

1. **Generate the logo** using one of the prompts above in ChatGPT/DALL-E
2. **Save it** as `logo.png` or `logo.svg` in the `/static/` folder
3. **Test it** by running `npm run dev` and checking the navigation bar
4. **Adjust if needed** - the logo should be visible and look professional

## Design Tips

- Keep it simple and recognizable at small sizes
- Ensure text is readable
- Use colors that match your site (blue theme)
- Make sure it works on both light and dark backgrounds (if you plan to use dark theme navigation)
